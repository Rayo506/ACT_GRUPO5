Notas para Admin / Datos

Ideas de integración y flujos:

  • Datos productos: DynamoDB (clave = productId). Backup con S3 o snapshots.
  • Imágenes y assets: almacenar en S3 + servir vía CloudFront.
  • API: API Gateway -> Lambda (Node/Python) que haga CRUD sobre DynamoDB.
  • Autenticación: Cognito para usuarios; IAM roles para Lambdas y accesos restringidos.
  • Billing / pagos: integrar Stripe en backend (no desde frontend directamente sin servidor).
  • CI/CD: AWS Amplify o GitHub Actions -> despliegue a S3/CloudFront.
